# Always Remember Me v4.0 产品需求文档（PRD）
> 产品目标：简洁好用的 AI 小说续写工具

---

## 一、核心目标

**Always Remember Me** 是一款专注于**小说续写**的工具 App，通过 AI 知识图谱技术帮助作者在续写时保持人物设定一致、伏笔连贯、剧情逻辑自洽。

核心体验目标：
- **简洁** — 用户进来，5秒内开始续写
- **专注** — 不被无关功能干扰，核心流程一步到位
- **可信赖** — 续写质量有前置校验保障，图谱让作者对人物关系了如指掌

---

## 二、目标用户

- 网络小说作者（尤其是日更作者，需要快速续写）
- 有长篇创作需求的文学爱好者
- 需要在续写时保持设定一致性的任何创作者

---

## 三、用户核心路径

```
打开 App
    ↓
    ├── 首次使用 → 导入小说 → 自动解析章节 → 生成图谱 → 开始续写
    │
    └── 日常使用 → 选续写基准章节 → 开始续写 → 预览/调整 → 加入链条
```

**交互规则：**
- 状态概览默认展开，点击标题栏可折叠

- 质量评估和前置校验：仅在有问题（未通过）时展开详情，通过时只显示简要状态
- 批量续写：BottomSheet 选择起始章节，而不是嵌套 AlertDialog

## 五、功能列表

### 5.1 小说管理
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 导入 TXT 小说 | 自动解析章节标题和内容 | P0 |
| 多书籍书架 | 切换当前书籍 | P0 |
| 删除书籍 | 确认后删除所有数据 | P1 |
| 阅读进度 | 自动记录上次阅读章节 | P1 |

### 5.2 续写核心
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 选择续写基准章节 | Dropdown 选章节 | P0 |
| 编辑基准章节内容 | 续写前可微调原文 | P0 |
| AI 续写生成 | 调用 API 生成续写内容 | P0 |
| 续写预览 | 显示生成结果 | P0 |
| 复制到剪贴板 | 一键复制续写内容 | P1 |
| 前置校验 | 人设红线、禁区规则、伏笔引用 | P0 |
| 质量评估 | 续写完成后打分 | P1 |
| 续写链条 | 管理多条续写结果 | P0 |
| 基于续写继续写 | 以某条续写为基础继续生成 | P0 |
| 批量续写 | 从起始章节自动向后续写全部 | P1 |
| 停止续写 | 中断正在进行的生成 | P1 |

### 5.3 知识图谱
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 单章图谱生成 | 为选定章节生成知识图谱 | P0 |
| 批量图谱生成 | 为所有章节生成图谱 | P1 |
| 分批合并图谱 | 每批50章合并 | P1 |
| 全局图谱合并 | 所有章结合并为全局图谱 | P0 |
| 查看全局图谱 | 阅读合并后的完整图谱 | P0 |
| 图谱导出 JSON | 复制图谱数据 | P2 |
| 图谱导入 JSON | 粘贴导入图谱 | P2 |

### 5.4 阅读器
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 章节翻页 | 左右滑动或点击翻页 | P0 |
| 进度滑块 | 拖动跳转章节 | P0 |
| 字号调节 | 12-28号调节 | P1 |
| 章节目录 | 从目录页跳转章节 | P0 |
| 阅读进度保存 | 自动保存退出位置 | P0 |

### 5.5 设置
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 前置校验开关 | 是否启用续写前的人设检查 | P2 |
| 批量续写重试次数 | 失败后自动重试次数 | P2 |

## 六、非功能需求

| 需求 | 标准 |
|------|------|
| 启动速度 | 数据加载时显示 loading，3秒内可操作 |
| 图谱生成可中断 | 停止按钮有效 |
| 批量续写可中断 | 停止按钮有效，失败自动重试3次 |
| 防空回检测 | 续写 < 50字自动标记失败 |
| 数据持久化 | 关闭重启后所有数据正确恢复 |
| JSON 安全 | 导入时校验格式，防 XSS/注入 |
