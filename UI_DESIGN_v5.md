## 1. 整体 UI 架构

### 1.1 整体结构（延续 v4.0 BottomNav 3Tab）

```
┌─────────────────────────────────────────────────────┐
│                    状态栏（系统）                     │
├─────────────────────────────────────────────────────┤
│  AppBar: [☰] 书名                    [阅读] [⋮]    │
│           (每个 Tab 各自持有)                        │
├─────────────────────────────────────────────────────┤
│                                                     │
│              当前 Tab 内容区                          │
│        Tab1: 续写 / Tab2: 阅读 / Tab3: 书架          │
│                                                     │
├─────────────────────────────────────────────────────┤
│     [✍️ 续写]      [📖 阅读]      [📚 书架]        │
│        Tab1           Tab2           Tab3           │
└─────────────────────────────────────────────────────┘

左侧滑出 Drawer（5项）：
┌──────────────────────────────────────┐
│  🐋 Always Remember Me                │
│  ═══════════════════════════════════  │
│  当前：《小说名》                      │
├──────────────────────────────────────┤
│  📖 章节管理                   [→]    │
│  🌲 全局图谱                   [→]    │
│  📤 导入/导出图谱              [→]    │
│  ⚙️ 续写参数                   [→]    │
│  ────────────────────────────────────  │
│  📱 设置                        [→]    │
└──────────────────────────────────────┘
```

### 1.2 路由架构（GoRouter）

```
/                           → 重定向到 /write（Tab1）
/write                      → WriteScreen（Tab1 续写）
/write/settings             → WriteSettingsScreen（续写参数）
/reader                     → ReaderScreen（Tab2 阅读）
/reader/:chapterId          → ReaderScreen（指定章节）
/bookshelf                  → BookshelfScreen（Tab3 书架）
/bookshelf/novel/:novelId   → NovelDetailScreen（小说详情）
/chapters                   → ChaptersScreen（Drawer 入口）
/chapters/import            → ImportScreen（导入小说）
/chapters/:chapterId/edit   → ChapterEditScreen（编辑章节）
/graph                      → GraphViewerScreen（Drawer 入口）
/graph/merge                → GraphMergeScreen（图谱合并）
/graph/import-export         → GraphImportExportScreen（导入导出）
/settings                   → SettingsScreen（Drawer 入口）
```

### 1.3 BottomNav 布局

| Tab | 图标 | 标签 | 路由 | 职责 |
|-----|------|------|------|------|
| Tab1 | ✍️ | 续写 | `/write` | 续写操作 + 续写链条 + 前置校验 + 质量评估 |
| Tab2 | 📖 | 阅读 | `/reader` | 小说阅读器 + 章节导航 + 字号/主题设置 |
| Tab3 | 📚 | 书架 | `/bookshelf` | 小说列表 + 导入 + 小说详情 |

### 1.4 Drawer 菜单（5项 → 6项含分隔）

| # | 菜单项 | 图标 | 路由 | 说明 |
|---|--------|------|------|------|
| 1 | 章节管理 | 📖 | `/chapters` | 图谱生成、批量生成、章节状态 |
| 2 | 全局图谱 | 🌲 | `/graph` | 合并后的完整知识图谱 |
| 3 | 导入/导出图谱 | 📤 | `/graph/import-export` | JSON 格式管理 |
| 4 | 续写参数 | ⚙️ | `/write/settings` | AI 模型、字数、前置校验开关 |
| 5 | ──────── | ─ | ─ | 分隔线 |
| 6 | 设置 | 📱 | `/settings` | API 配置、阅读器设置、关于 |

### 1.5 页面跳转流程

```
┌─────────────────────────────────────────────────────────────────┐
│                        BottomNav 3 Tabs                         │
├─────────────────┬─────────────────┬─────────────────────────────┤
│   Tab1: 续写     │   Tab2: 阅读     │   Tab3: 书架                 │
│   /write        │   /reader        │   /bookshelf                │
└────────┬────────┴────────┬────────┴──────────────┬────────────────┘
         │                │                       │
         │                │                       │
         ▼                ▼                       ▼
   Drawer → 章节管理   Drawer → 全局图谱    Drawer → 章节管理
   /chapters           /graph              /chapters
         │                │                       │
         │                │                       │
         ▼                ▼                       ▼
   导入小说          图谱合并               小说详情
   /chapters/import  /graph/merge         /bookshelf/novel/:id
                          │
                          ▼
                    导入/导出
                    /graph/import-export
```

**关键跳转规则：**
- Tab 内跳转：GoRouter push（保留 BottomNav）
- Drawer 跳转：GoRouter push（保留 BottomNav）
- 全屏跳转（如导入）：GoRouter push（覆盖）
- 返回：Navigator.pop 或 AppBar 返回按钮
