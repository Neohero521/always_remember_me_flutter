## 1. 整体布局设计

### 1.1 整体结构

```
┌─────────────────────────────────────────────┐
│              状态栏（系统）                   │
├─────────────────────────────────────────────┤
│  AppBar: [☰] 《书名》        [📖] [⋮]         │  ← 每个 Tab 各自持有
├─────────────────────────────────────────────┤
│                                             │
│              当前 Tab 内容区                 │
│          （Tab1: 续写 / Tab2: 阅读 / Tab3: 书架）│
│                                             │
├─────────────────────────────────────────────┤
│  [✍️ 续写]    [📖 阅读]    [📚 书架]         │  ← BottomNavigation，固定底部
└─────────────────────────────────────────────┘

左侧滑出：
┌──────────────────────────┐
│ 🐋 Always Remember Me     │
│ 当前：《小说名》            │
├──────────────────────────┤
│ 📖 章节管理        →      │
│ 🌲 全局图谱        →      │
│ 📤 导入/导出图谱   →      │
│ ⚙️ 续写参数        →      │
│ ───────────────────────── │
│ 📱 设置            →      │
│ ℹ️ 关于            →      │
└──────────────────────────┘
```

### 1.2 导航架构

| Tab | 图标 | 标签 | 对应文件 | 职责 |
|-----|------|------|----------|------|
| Tab1 | ✍️ | 续写 | `write_screen.dart` | 续写操作 + 续写链条管理 |
| Tab2 | 📖 | 阅读 | `reader_screen.dart` | 小说阅读器 |
| Tab3 | 📚 | 书架 | `bookshelf_screen.dart` | 书籍管理（导入、切书、删除） |

### 1.3 Drawer 结构（5 项）

| # | 菜单项 | 图标 | 跳转目标 |
|---|--------|------|----------|
| 1 | 章节管理 | 📖 | `ChaptersScreen` |
| 2 | 全局图谱 | 🌲 | `GraphViewerScreen` |
| 3 | 导入/导出图谱 | 📤 | `GraphImportExportScreen` |
| 4 | 续写参数 | ⚙️ | `WriteSettingsScreen` |
| 5 | 设置 | 📱 | `SettingsScreen` |
| 6 | 关于 | ℹ️ | AboutDialog |

### 1.4 页面切换动效

| 交互 | 动效 | 时长 |
|------|------|------|
| BottomNav Tab 切换 | Fade + Slide | 200ms |
| Drawer 滑出 | 从左侧边缘滑入，半透明遮罩渐显 | 250ms |
| Drawer 关闭 | 向左滑出，遮罩渐隐 | 200ms |
| 续写结果展开 | 高度动画 + 滚动 | 300ms |
| 状态卡片折叠 | 高度动画 | 200ms |
| 批量续写 BottomSheet | 从底部滑入 | 250ms |
| 章节选择下拉 | Flutter 原生 Dropdown 动效 | 原生 |

### 2.2 AppBar 设计

| 元素 | 说明 |
|------|------|
| 左侧 Drawer 按钮 | ☰ 图标，点击打开 Drawer |
| 中间书名 | 当前书籍名称，无书时显示「未选择书籍」 |
| 右侧按钮1 | 📖 阅读快捷入口，无书时禁用（`enabled: false`） |
| 右侧按钮2 | ⋮ 更多菜单（续写参数、设置） |

### 2.3 状态概览卡片（CollapsibleCard）

**展开状态：**
```
┌─────────────────────────────────────────────┐
│  📊 状态概览                       [▼ 折叠] │
├─────────────────────────────────────────────┤
│  [📖 12章节] [🌲 8图谱] [✍️ 3续写] [✅已合并] │
└─────────────────────────────────────────────┘
```

**折叠状态：**
```
┌─────────────────────────────────────────────┐
│  📊 状态概览                       [▶ 展开] │
└─────────────────────────────────────────────┘
```

**状态项说明：**
| 状态项 | 图标 | 颜色 | 含义 |
|--------|------|------|------|
| 章节数 | 📖 | Primary | 总章节数量 |
| 图谱数 | 🌲 | Primary | 已生成图谱的章节数 |
| 续写数 | ✍️ | Accent | 续写链条中的条目数 |
| 合并状态 | ✅/⏳ | Green/Amber | 全局图谱是否已合并 |

### 2.4 续写基准章节选择区

**无图谱时（警告状态）：**
```
┌─────────────────────────────────────────────┐
│  ⚠️ 该章节暂无图谱，建议先生成后再续写        │
│  [🌲 立即生成图谱]                          │
└─────────────────────────────────────────────┘
```

**Dropdown 样式：**
- 触发区域高度：56px
- 每项显示：章节序号 + 章节名（截断）+ 字数
- 无书籍时：显示「请先导入小说」

### 2.5 操作按钮区

| 按钮 | 样式 | 优先级 |
|------|------|--------|
| 🚀 开始续写 | 主按钮，高度 52px，Full Width，Primary Color | P0 |
| ⚡ 批量续写 | 次要按钮，Outlined，40px 高 | P1 |
| 📥 导入小说 | 次要按钮，Text Button，40px 高 | P1 |

**批量续写 BottomSheet：**
```
┌─────────────────────────────────────────────┐
│  ══════════ 批量续写设置 ══════════         │
├─────────────────────────────────────────────┤
│  起始章节                                     │
│  [第3章 夜色降临                        ▼]  │
│                                             │
│  结束章节（留空 = 续写到最后一章）             │
│  [第5章                          ▼] [留空]  │
│                                             │
│  自动重试失败章节：                           │
│  [  ● 关闭  ○ 1次  ○ 3次 ]                 │
│                                             │
│  [取消]              [🚀 开始批量续写]        │
└─────────────────────────────────────────────┘
```

### 2.6 续写结果区（折叠面板）

**折叠时（无结果）：** 不渲染任何内容

**折叠时（有结果但未展开）：**
```
┌─────────────────────────────────────────────┐
│  ✨ 续写完成 · 约 1200 字         [▶ 展开]  │
└─────────────────────────────────────────────┘
```

**展开状态：** 见 2.1 完整布局

**质量评估颜色规范：**
| 分数区间 | 颜色 | 标签 |
|----------|------|------|
| 90-100 | Green | 优秀 |
| 70-89 | Primary | 良好 |
| 50-69 | Amber | 一般 |
| < 50 | Red | 较差 |

**前置校验状态颜色：**
| 状态 | 颜色 | 图标 |
|------|------|------|
| 通过 | Green | ✅ |
| 警告 | Amber | ⚠️ |
| 失败 | Red | ❌ |

### 2.7 续写链条列表

**空状态：**

```
┌─────────────────────────────────────────────┐
│  ✍️ 续写链条                       [全部清空]│

│  暂无续写记录，快去开始你的第一次续写吧~ 🐋    │

```

**列表项：**
| 字段 | 说明 |
|------|------|
| 🔗 图标 | 固定显示 |
| 标题 | 「续写#N · 第X章续写」 |
| 副标题 | 字数 + 生成时间 |
| 操作按钮 | [→继续] [📋] [🗑] |

### 4.2 书籍卡片设计

**当前书籍高亮样式：**
- 左侧 4px 宽 Primary 颜色边框
- 背景：Primary 颜色 5% 透明度
- 「当前」标签显示在右上角

**普通书籍：**
- 无左边框高亮
- 标准背景色

**书籍卡片内容：**
| 字段 | 示例 |
|------|------|
| 封面 | 占位图或网络封面，60×80px |
| 书名 | 单行，超长截断 |
| 章节数 | 「12章」 |
| 上次阅读 | 「上次阅读：2天前」/「从未阅读」 |
| 操作 | 阅读、续写、删除 |

### 4.3 空状态

### 4.4 快捷操作条（无书时隐藏）

- 点击任意区域切换书籍（点击当前书籍卡片内区域也可）
